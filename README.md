# Executive AI Readiness Copilot

A portfolio-quality MVP web app that helps executives assess AI readiness, identify high-value AI opportunities, and generate a practical action roadmap.

## Project Goal

Build a polished executive-facing app that demonstrates:
- product judgment
- scope discipline
- clean UI
- practical AI integration
- credible MVP execution

## MVP Scope

This MVP includes:
- landing page
- AI readiness assessment
- deterministic scoring logic
- results dashboard
- top opportunity recommendations
- 30/60/90-day roadmap
- executive summary
- copy/export-ready output

This MVP does not include:
- authentication
- billing
- collaboration
- admin tools
- heavy backend architecture

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vercel

## Local Development

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
